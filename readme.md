# Webtools Lite Extension

각종 웹 툴을 모아놓은 확장 프로그램입니다.

기능

- 글자 수 세기(완료)
- 대소문자 변환(완료)
- 단위 변환(완료)
- 진수 변환(완료)
- 한영타 변환(완료)
- 통합 주소 검색(완료)
- 색 추출(완료)

개발 스택

- TypeScript(메인 기능 작성)
- Vite(번들링)
- Cloudflare(api 응답)
