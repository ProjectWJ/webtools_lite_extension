# Webtools Lite Extension

각종 웹 툴을 모아놓은 확장 프로그램입니다.

기능

- 글자 수 세기(구현)
- 대소문자 변환(구현)
- 단위 변환
- 진수 변환
- 한영타 변환
- 환율 계산
- 날짜 계산
- 도로명-지번 변환

개발 스택

- TypeScript
- Vite
